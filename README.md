# 📊 Employee Payroll System with Data Visualization

An automated payroll management system built with Python using **Object-Oriented Programming (OOP)** principles. This application calculates employee salaries, handles tax deductions, and provides a visual breakdown of department-wise expenses using **Matplotlib**.

---

## 🚀 Key Features

* **Inheritance & Polymorphism:** Implementation of a base `Employee` class with specialized `SalariedEmployee` and `HourlyEmployee` subclasses.
* **Dynamic Tax Calculation:** Automated tax logic based on gross salary brackets.
* **Data Aggregation:** Calculates total net payout for different company departments (IT, HR, Finance).
* **Data Visualization:** Generates professional bar charts to help management visualize payroll distribution.

---

## 🛠️ Tech Stack

| Technology | Usage |
| :--- | :--- |
| **Python** | Core logic and OOP implementation |
| **Matplotlib** | Data visualization and Chart generation |
| **Git** | Version control |

---

## 📈 Salary Structure Logic

The system follows a progressive tax bracket:
- **Salary ≤ 50,000:** 10% Tax
- **Salary ≤ 100,000:** 20% Tax
- **Salary > 100,000:** 30% Tax
